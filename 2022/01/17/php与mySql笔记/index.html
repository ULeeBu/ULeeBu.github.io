php与mySql笔记

php的介绍

php必须在阿帕奇服务器下运行,将运行的代码放在www目录下.
www目录也称为部署目录.
访问时www目录就是本机ip地址.

比如有一个index.html文件,访问时需要通过
本机ip/index.html方式访问

* php文件的作用
  1.处理后端数据
  2.像HTML一样渲染页面（几乎没用）

* php的代码必须用<?php
                ?>
                包含起来
* php支持中文的函数
  header("Content-type:text/html;charset=utf-8");

* php的每行语句必须加分号

* echo等价于document.write

* 注释的三种方式
  // /**/ #

* 环形地址代表本机IP:
  127.0.0.1
  localhost

数据库

操作数据库用的是sql语句

sql编辑器:用来编写sql语句的

如何使用sql编辑器:一定要选中需要操作的库

创建一个表

create table 表名 (字段1,字段2…字段N);
create table student
(
stu_id int,
stu_name varchar(10),
stu_gender char(5),
stu_date date
)

增
insert into 表名 [字段1,字段2…字段N]
values (值1,值2…值n)
insert into student (stu_id,stu_name,stu_gender,stu_date)
values (3,”老王”,”M”,”2021-11-22”);
insert into student
values (4,”小明”,”M”,”2022-1-2”);

删表
drop table 表名;
drop table student;

删记录
delete from 表名;
delete from student;

实际情况是要根据条件删除某行记录
where子句可以实现条件的筛选
delete from student
where stu_id = 1;

AND 与
OR 或
delete from student
where stu_name = “老王”
AND stu_id = 3;

改
update 表名 set 字段1=值1,字段2=值2…
update student set stu_gender = “FM”;
update student set stu_name = “董文博”
where stu_id = 1;

查
select 字段1,字段2…from 表名;
select stu_id,stu_name from student;
select * from student;
select * from student
where stu_id = 1;
